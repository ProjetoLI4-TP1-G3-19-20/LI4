O servidor de desenvolvimento, localizado na pasta "LI4", NÃO ESTÁ ATUALIZADO. 



A pasta chamada LI4 contém o código que foi usado para desenvolvimento do servidor, numa máquina local.

No entanto, quando foi necessário correr o código numa VM, foi preciso criar um novo projeto de VS com os mesmo ficheiros fonte, pois os projetos de VS dependem dos caminhos absolutos para os ficheiros.
